# tp-gts

## 依赖安装
```
$ npm add gulp-cli -g
$ npm add typescript ts-node gulp gulp-typescript -D
$ npm add lodash
$ npm add browserify tsify vinyl-source-stream -D
$ npm add watchify fancy-log -D
$ npm add gulp-uglify vinyl-buffer gulp-sourcemaps -D
$ npm add babelify@8 babel-core babel-preset-es2015 -D
```
